# Statum CRM
## Statum — это современная бэкенд-система управления взаимоотношениями с клиентами (CRM), ориентированная на автоматизацию работы с контрагентами, сделками и историей взаимодействий. Проект разработан как программная реализация дипломной работы.

## Основные возможности

- Управление контрагентами: Регистрация клиентов с автоматической проверкой уникальности ИНН и валидацией контактных данных.

- Учет сделок: Полноценная воронка продаж с отслеживанием сумм и текущих статусов (NEW, NEGOTIATION, PAID, CLOSED).

- Журнал активностей: Фиксация истории звонков, встреч и переписки с клиентами для обеспечения непрерывности процесса продаж.

- Аналитический модуль: Автоматическая агрегация финансовых показателей по всем этапам воронки продаж.

- Документированный API: Полная поддержка спецификации OpenAPI (Swagger) для упрощения интеграции с фронтендом.

## Технологический стек
- Язык программирования: Java 21

- Фреймворк: Spring Boot 3.3 (Web, Data JPA)

- База данных: PostgreSQL 18

- ORM: Hibernate

- Маппинг: MapStruct (для разделения Entity и DTO)

- Инструменты: Lombok (минимизация бойлерплейт-кода)

- Документация: SpringDoc OpenAPI (Swagger UI)

- Тестирование: JUnit 5, Mockito, MockMvc

## Запуск и развертывание
### Предварительные требования
- Установленная JDK 21.
- Запущенный экземпляр PostgreSQL.

### Установка
```
git clone https://github.com/your-username/statum-crm.git
```
Настройте параметры подключения к БД в src/main/resources/application.yaml и соберите проект с помощью Maven:
```
mvn clean install
```
Запустите приложение:
```
mvn spring-boot:run
```
